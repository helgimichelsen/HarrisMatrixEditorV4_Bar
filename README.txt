Harris Matrix Editor 4.0 BAR

Auto-layout er baseret på BAJR Guide 23, kapitlet "Stratigraphy and Harris Matrix":
- Matricen viser relativ stratigrafisk rækkefølge, ikke udgravningsrækkefølge.
- Kæd kun det man ved; antag ikke at ens lag er samme kontekst.
- Ukoblede men lignende lag holdes som parallelle grene.
- Direkte samme kontekst kan markeres med =.
- Natural/Unexcavated placeres nederst.
- Topsoil/T placeres øverst.
- Relationer gemmes som yngre/øverst -> ældre/nederst.

Funktioner:
- Stiplede gruppe-/konstruktionsbokse kan flyttes og skaleres.
- Auto-layout BAR/Harris.
- Import relationstekst.
- Import HMCX (generisk XML/HMCX-parser; kontrollér resultatet bagefter).
- Eksport SVG.
- Eksport PDF.
- Harris-kontrol og rapport.
